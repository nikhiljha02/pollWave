# PollWave 🌊

A real-time polling and voting platform built using the MERN stack with Socket.IO for live updates.

## 🚀 Live Demo

Frontend: https://poll-wave-psi.vercel.app  
Backend: https://pollwave-0qpi.onrender.com

---

# ✨ Features

- 🔐 User Authentication
- 📊 Create and Manage Polls
- 🗳 Real-time Voting
- ⚡ Live Updates using Socket.IO
- 📱 Responsive UI
- 🌐 MERN Stack Architecture
- 🔄 Dynamic Dashboard
- ☁️ Cloud Deployment

---

# 🛠 Tech Stack

## Frontend
- React
- Vite
- React Router
- Axios
- Socket.IO Client

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Socket.IO

## Deployment
- Frontend → Vercel
- Backend → Render
- Database → MongoDB Atlas
